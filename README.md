## Deskripsi
Repositori ini adalah repositori website lokal untuk menyelesaikan permasalahan chinese remainder problem beserta menampilkan langkah-langkah penyelesaiannya.

## Cara Penggunaan Program
1. Buka terminal arahkan ke direktori tempat program disimpan yaitu pada folder 'Chinese-Remainder-Solver' atau nama folder tempat repository ini disimpan.
2. Jalankan perintah berikut untuk menjalankan program.
```
npm run start
```
3. Setelah itu buka localhost yang ditampilkan pada program sehingga web lokal dapat diakses
